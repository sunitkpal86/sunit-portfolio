HTML

HTML stands for Hypertext Markup Language and is the language for creating web pages

It is a markup language and not a programming language

We “mark” certain things in the code we write which tells the web browser how to structure the web page

It does not contain any logic like conditionals or branching

HTML consists of a series of elements and each element tells the browser how to display the content

HTML5 is the fifth and latest version of HTML

Advantages of HTML5

New Semantic Elements: header, nav, article, section, aside, footer

Form Input Types : input type=”date”>, input type = “email”>

Audio and Video Support : audio>, video>

Mobile Compatibolity

Simpler Code

XHTML

XHTML (eXtensible HyperText Markup Language) is a markup language that follows the rules of XML to define the structure of web pages.

HTML/HTML5

HTML has more lenient syntax rules. Closing tag is not mandatory for some elements. ima src=”sunit.jpg” alt=”Description”>.

HTML is not case-sensitive. DIV>, P> will work.

HTML is widely supported by all browsers and web platforms.

XHTML

XHTML follows stricter syntax rules. All tags must be properly nested and closed. img src=”sunit.jpg” alt=”Description” />

XHTML is case-sensitive. div>, p> will work.

XHTML has limited support by browsers.

title>

The title> tag in HTML is used to define the title of a web page.

Advantages of title tag:

Window Title : title> element text is displayed in the tab of the browser.

SEO : Search engines use the title as the main heading for search results.

Bookmarks/Favorite : Title tag is used as the default name for the bookmark.

Social Media Sharing : Title tag is used as the default title in the shared post.

Meta tags

Meta tags in HTML are elements used to provide metadata or additional information about a web page.

Types of Meta Tags

charset

viewport

description

keywords

author

head>
     meta charset="UTF-8">
     meta name="viewport" content="width=device-width, initial-scale=1.0">
     title>Sunit Online/title>
     meta name="description" content="Technical Subjects" />
     meta name="keywords" content=".NET, Angular, Reactjs, HTML" />
     meta name="author" content="Sunit Pal" />
/head>

Character Encoding :

This helps browsers to interpret the different characters in the document. The charset=”UTF-8” attribute ensures that characters from different languages will be displayed correctly in the browser.

Responsive Design :

viewport meta tag is crucial for creating mobile-friendly, responsive web designs for various screen sizes.

Description for SEO :

This meta tag provides description of the content of the page. Search engines may use this for search results.

Keywords for SEO :

This meta tag used to be important for search engines, but it’s now of less importance.

Author Info :

This meta tag can be used to specify the author of the document.

HTML Elements & Tags

HTML elements are the building blocks of web pages.

Element consists of a start tag, content, and an end tag.

Tag is a specific part of an element that denotes the beginning or end of that element.

Most common used HTML elements

Paragraph Element : Defines a block of text. p>

Heading Elements : h1>h2>h3>h4>h5>h6>

Header and Footer Elements : header> and footer>

Sectioning Elements : section> article>, nav>, aside>, main>

Image Element : Embeds images into a webpage. img> (self-closing).

List Elements : ul> (unordered list) , ol> (ordered list) , li> (list item)

Anchor Element (Link) : Creates hyperlinks to other pages or resources. a>.

Division Element : div> groups together content for applying styles or scripting. div>.

Table Elements : table>, tr> (table row), th> (table header) and td> (table data).

Form Elements : Used for creating forms. form>, input>, button>, select> , textarea> , etc.
